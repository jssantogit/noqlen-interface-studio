# Anchor Visual References

Expected reference image:

- `anchor-screens-reference.png`

Reference images in this directory are design targets for Anchor UI implementation. They are not runtime assets, should not be imported by app code and should not ship as part of the app UI.

Anchor screens should be implemented with high visual fidelity to the provided reference image. Do not reinterpret the target into generic placeholder panels, generic mobile cards or unrelated layout patterns.

All Anchor behavior remains mock-only. Do not add real backend calls, server control, library access, downloads, playback, personal paths or app integration.
