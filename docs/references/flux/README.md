# Flux Visual References

Expected reference image:

- `flux-screens-reference.png`

Reference images in this directory are committed documentation/spec assets and visual design targets for Flux UI implementation. They are not runtime app assets, should not be imported by app code and should not ship as part of the app UI.

Flux screens should be implemented with high visual fidelity to the provided reference image. Do not reinterpret the target into generic placeholder panels, generic workflow dashboards or unrelated layout patterns.

All Flux behavior remains mock-only. Do not add real backend calls, workflow execution, server control, library access, downloads, playback, personal paths or app integration.
