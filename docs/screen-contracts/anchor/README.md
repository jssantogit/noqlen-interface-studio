# Anchor Screen Contract

Anchor is the default tab for Bloco 0.

Required qualities:

- More developed than Flux, Forge and Aria.
- Shows homepage shell direction.
- Includes mock status indicators.
- Communicates visual-only boundaries.
- Works on desktop and mobile.

Forbidden behavior:

- Real app launch.
- Real downloads.
- Server or process control.
- Reading local libraries or personal paths.
