# Aria Visual Target

## Primary visual references

- `docs/references/aria/aria_reference1.png`
- `docs/references/aria/aria_reference2.png`

These supersede all previous Aria mockups.

Goal: implement Aria as an immersive music player/library mock UI.

Visual rules:

- Preserve large album art.
- Preserve the now playing screen.
- Preserve lyrics/player state.
- Preserve artist profile treatment.
- Preserve library categories.
- Preserve the mini player.
- Preserve playlists/stations.
- Preserve a warm dark premium mood.
- Preserve amber accents.
- Preserve the image-heavy music UI.

Must not:

- Perform real playback.
- Access a real music library.
- Fetch metadata.
- Download files.
- Stream audio.

All Aria behavior remains mock-only.
