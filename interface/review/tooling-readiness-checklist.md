# Tooling Readiness Checklist

- `node`, `npm` and `npx` are available, or limitation is reported.
- `opencode` is available, or limitation is reported.
- Browser MCP config is present locally, or limitation is reported.
- Playwright MCP is available, or limitation is reported.
- Chrome DevTools MCP is available, or limitation is reported.
- Serena availability is checked.
- Serena read-only mode is documented before use.
- RTK availability is checked.
- Context Mode availability is checked.
- Caveman is disabled or not configured.
- Local tool config files are ignored or untracked.
- No local generated tool state is tracked.
- Tool Mode is declared in the block prompt/report.
